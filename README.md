# Amrita Attendance Fetcher

A Chrome extension that enhances the attendance tracking experience for Amrita University students.

## Features

- Automatically fetches attendance data from the university portal
- Redirects to a comprehensive attendance dashboard
- Provides improved visualization of attendance metrics
- Helps students track their attendance requirements easily

## Installation

1. Download the extension from the [Chrome Web Store](https://chrome.google.com/webstore)
2. Click "Add to Chrome" to install
3. Login to your university portal as usual

## Usage

1. Navigate to the university attendance portal
2. The extension will automatically fetch your attendance data
3. You'll be redirected to the enhanced attendance dashboard
4. View detailed attendance statistics and visualizations
